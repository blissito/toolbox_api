DuckDuckGo Search - Documentación

Realiza búsquedas en la web utilizando el motor de búsqueda DuckDuckGo de forma anónima y sin restricciones.

Otras herramientas disponibles:

WebFetch DuckDuckGo Search

Uso Básico

Realiza una búsqueda en DuckDuckGo y obtén los resultados en formato JSON.

Endpoint

POST /api/tool

Parámetros

Parámetro Tipo Requerido Descripción
query string Sí Término de búsqueda
max_results number No Número máximo de resultados a devolver (por defecto: 10)
region string No Región para los resultados (ej: 'wt-wt' para internacional, 'es-es' para España)

Ejemplo de solicitud

curl -X POST \
  https://toolbox-api.fly.dev/api/tool \
  -H 'Content-Type: application/json' \
  -d '{
    "tool": "duckduckgo_search",
    "query": "inteligencia artificial",
    "max_results": 5,
    "region": "es-es"
  }'

Códigos de Estado

Código Descripción
200 OK - Búsqueda completada con éxito
400 Solicitud incorrecta (faltan parámetros obligatorios)
429 Demasiadas solicitudes (límite de tasa excedido)
500 Error interno del servidor
