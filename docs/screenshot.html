Screenshot Tool

La herramienta de captura de pantalla te permite tomar capturas de cualquier página web de forma programática. Es perfecta para generar vistas previas, monitorear cambios en sitios web o crear miniaturas de páginas.

Endpoint

POST /api/tool

Toma una captura de pantalla de una URL especificada.

Parámetros

Parámetro Tipo Requerido Descripción
tool string Sí Debe ser "screenshot"
payload.url string Sí URL de la página de la que se tomará la captura

Autenticación

Este endpoint requiere autenticación mediante API Key.

Incluye tu API Key en el encabezado de la solicitud:

Authorization: Bearer tbx_tu_api_key_aquí

Ejemplo de Uso

Solicitud

POST /api/tool
Content-Type: application/json
Authorization: Bearer tbx_tu_api_key_aquí

{
    "tool": "screenshot",
    "payload": {
        "url": "https://example.com"
    }
}

Respuesta Exitosa (200 OK)

La respuesta es una imagen PNG que puedes mostrar o guardar.

Content-Type: image/png

[datos binarios de la imagen]

Errores Comunes

400 Bad Request

Falta el parámetro URL o es inválido.

{
    "success": false,
    "error": "Se requiere el parámetro 'url' en el payload",
    "code": "missing_url"
}

401 Unauthorized

API Key inválida o faltante.

{
    "success": false,
    "error": "No autorizado: formato de clave API inválido",
    "code": "unauthorized"
}

500 Internal Server Error

Error al procesar la captura de pantalla.

{
    "success": false,
    "error": "Error al tomar la captura de pantalla: [mensaje de error]",
    "code": "screenshot_failed"
}

Ejemplo en JavaScript

async function takeScreenshot(apiKey, url) {
    const response = await fetch('https://api.toolbox.com/api/tool', {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json',
            'Authorization': `Bearer ${apiKey}`
        },
        body: JSON.stringify({
            tool: 'screenshot',
            payload: { url }
        })
    });

    if (!response.ok) {
        const error = await response.json();
        throw new Error(error.error || 'Error al tomar la captura de pantalla');
    }

    // Si necesitas mostrar la imagen
    const blob = await response.blob();
    const imageUrl = URL.createObjectURL(blob);
    
    // O si necesitas descargarla
    const downloadLink = document.createElement('a');
    downloadLink.href = imageUrl;
    downloadLink.download = 'screenshot.png';
    downloadLink.click();
    
    return imageUrl;
}
